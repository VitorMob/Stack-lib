# Stack

## Help

for usage stack created obj `Stack stack(10)` and allocate 10, for stack `stack_push(0x15)` 

for unstack `stack_pop()` 

for get bit significative `stack_sign()`

for get size stack `stack_size()`

> example

```C++

  Stack stack(10);

  stack.push(0x15);
  printf("pop=%u", stack.stack_pop());

```
